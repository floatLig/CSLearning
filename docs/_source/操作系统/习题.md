- 同一文件在不同的存储介质上______的组织形式。(      )

C.必定用不同  
D.无所谓

- 什么是操作系统（OS）？它的主要功能是什么？

> 答：操作系统是控制和管理计算机系统内各种硬件和软件资源、`有效地组织多道程序运行的系统软件（或程序集合）`，是用户与计算机之间的接口。  
> 操作系统应具备的五大基本功能，即存储管理、进程和处理机管理、文件管理、设备管理、`用户接口`。
